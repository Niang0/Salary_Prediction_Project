# 💰 Salary Prediction

## 📌 Description

Salary Prediction est une application de Machine Learning permettant de prédire le salaire d'un employé à partir de plusieurs caractéristiques telles que son âge, son expérience et son département.

Ce projet a été réalisé dans le but de démontrer la mise en œuvre complète d'un pipeline de Machine Learning, depuis la préparation des données jusqu'au déploiement d'une interface utilisateur avec Streamlit.

---

## 🎯 Objectifs

- Nettoyer et préparer un dataset RH.
- Encoder les variables catégorielles.
- Entraîner un modèle de régression linéaire.
- Évaluer les performances du modèle.
- Sauvegarder le modèle entraîné.
- Développer une interface utilisateur avec Streamlit.
- Produire une architecture de projet professionnelle.

## ⚙ Installation

Cloner le projet :

git clone https://github.com/votre-utilisateur/salary-prediction.git

Installer les dépendances :

pip install -r requirements.txt

Lancer l'application :

streamlit run app/streamlit_app.py

---

## 📊 Résultats

Le modèle obtient les performances suivantes :

- MAE : 2 000 FCFA
- R² : 0,98

Ces résultats montrent que le modèle présente une faible erreur moyenne tout en expliquant environ 98 % de la variation des salaires observés.
